Easy2

Configuration files and supporting scripts for EC2 deployment

Download this project as a .zip file Download this project as a tar.gz file

Overview

easy2 is a collection of scripts and configuration files that aid the process of building and booting custom machine images (AMIs) on Amazon EC2.

Installation and usage

Here's how to install it. First, build and install the Debian package (on your EC2 instance):

$ git clone git://github.com/minmb/easy2.git
$ cd easy2
$ debuild -us -uc
$ sudo dpkg -i ../easy2*.deb

Then, set EC2_USER and EC2_GROUP in /etc/default/easy2 to the primary user/group for your AMI. Finally, run the init script:

$ sudo /etc/init.d/easy2 start

The ´ssh-key´ start hook will query the EC2 metadata API and fetch and append the public SSH key specified at instance creation to /home/$EC2_USER/.ssh/authorized_keys. You may install your own start/stop hooks in /etc/easy2/{start,stop}.d.
